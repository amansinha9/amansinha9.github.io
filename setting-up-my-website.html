Setting Up My Website

Aman Sinha | Thu 19 October 2017

Hello!

From this blog post, i want to share my first hand experience with pelican, a static site generator, written in Python. I got to know about this amazing site generator recently and i was quite amazed after knowing the features it provide. It has so many features which are pretty useful in making webpages. Some of its feature include: - Write you content in Markdown, AsciiDoc, reStructuredText formats. - Completly statin output is easy to host anywhere. - Publish content in multiple languages. - Code syntax highlighting.

I did play with it to know more and more about this and i found out that this is really a amazing site generator and very easy to learn. After sometime it will become handy.

Method to set up

To setup pelican in your desktop, first open the terminal and type the following command:

sudo install pelican markdown.

This command will install pelican in your pc.

The next step after installing would be creating a project. First choose the name for your project, and create an appropriately-named directory for your site, and switch to that directory.

mkdir -p ~/projects/yoursite
cd ~/projects/ yoursite

Enough for today. Will see you later.
