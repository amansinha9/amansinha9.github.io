* Data types:


  1. Numerical Data, quantifiable information that is measurable.

    • Discrete Data, integer based (eg. no.of people).
    • Continuous Data, decimal based (eg. price, distance).

  2. Categorical Data, qualitative data, used to classify data into categories. For example, gender, brands, etc.

  3. Ordinal Data, represents discrete and ordered units, e.g. league rank, hotel rating(1-5).


* Measures of Central Tendency:

Let's assume the dataset of 5 numbers:

{6, 3, 100, 3, 13}

  • Mean, average of the dataset:

mu = (6+3+100+4+13)/5 = 126/5 = 25

  • Median, middle number of the dataset:

100

  • Mode, common value in the dataset or which occurs more frequently:

3

* Measures of Variability:

  • Range, difference between the lowest and the hightest number of the dataset:

Range = 100-3 = 97

  • Variance, measures how spread out the data is, average of squared differences from the mean:
  • Find the mean of the data points

25

  1. Subtract the mean from each data point

6-25 = -19 3-25 = -22 100-25 = 75 3-25 = -22 13-25 = -12

  1. Square each result

(-19)^2 = 361 (-22)^2 = 484 (75)^2 = 5625 (-22)^2 = 484 (-12)^2 = 144

  1. Find the mean of the results

(361+484+5625+484+144)/5 = 1419.6

* Standard Deviation (sigma):

Just the square root of the variance.

sigma = sqrt(1419.6) = 37.68

* Data Distributions:

  1. Probability Distribution Function: Describes the likelihood of an event or outcome.

  2. Probability Density Function (PDF): Represents the probability of a given range of values occuring.

The area under the curve between two points corresponds to the probability that the variable falls between those two values.

  • Probability Mass Function (PMF): The probability of a given value occuring.
